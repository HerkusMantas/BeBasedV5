# **App Name**: Mind Weaver

## Core Features:

- Mind Map Display: Display an interactive mind map using SVG elements for nodes and connections.
- Node Creation: Allow users to create new nodes with text, type (main, sub, detail) and adjust properties such as color, size and position.
- Link Creation: Enable linking of nodes to represent relationships with customizable labels and styles.
- Node Manipulation: Allow the user to drag-and-drop nodes freely across the canvas.
- Save/Load/Export Mindmaps: Users can save their mind maps locally or export as JSON files to share.
- AI-Powered Concept Suggestions: Tool that suggests related concepts for selected node using a generative AI, which uses reasoning to decide if these suggested concept should be added to mind map or not.

## Style Guidelines:

- Primary color: Deep blue (#666666) to provide a sense of structure and clarity, mirroring the organized nature of mind maps.
- Background color: Very light blue (#999999), a desaturated version of the primary, provides a calming and unobtrusive backdrop for content creation.
- Accent color: Purple (#cccccc), provides contrast and signals important actions or highlighted nodes.
- Font: 'PT Sans', sans-serif, will be used for all text elements, ensuring readability and a clean user interface. This font will be used both for headlines and body text
- Use simple, clear icons to represent different node types and actions (e.g., plus sign for adding nodes, arrows for connections).
- A clean, minimalist layout that emphasizes the mind map itself, using a full-screen canvas to maximize creative space.
